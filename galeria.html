Sistemas Operativos

La diversidad es una de las mayores fortalezas del ecosistema Linux.

Kali Linux

Distribución líder en seguridad informática y pentesting.

Incluye herramientas como Metasploit, Nmap y Wireshark. Su entorno predeterminado es Xfce.

Parrot OS

Sistema operativo basado en Debian especializado en seguridad.

Ofrece herramientas para análisis forense digital y hacking ético. Su entorno Mate lo hace liviano.

Tsurugi Linux

Distribución especializada en análisis forense digital.

Incluye herramientas para adquisición y análisis de evidencia digital.

Ubuntu

Una de las distribuciones Linux más populares.

Con interfaz GNOME, disponible en versiones Desktop, Server y Core para IoT.

Arch Linux

Distribución ligera y flexible enfocada en la simplicidad.

Con instalación manual, proporciona una base mínima totalmente personalizable.

Debian

El pilar de numerosas distribuciones Linux.

Con tres ramas principales (Stable, Testing y Unstable) para diferentes necesidades.

Cada sistema operativo tiene su propio conjunto de fortalezas.
